# 🐍 贪吃蛇游戏 (Snake Game)

一个使用 HTML5 Canvas 和 JavaScript 实现的经典贪吃蛇游戏，具有现代化界面和多种游戏功能。

## 🎮 游戏特点

- **现代化界面**: 渐变背景、圆角设计、响应式布局
- **多种食物类型**: 
  - 🍎 普通食物 (+10分)
  - 🍒 加速食物 (+20分，加快蛇速)
  - 🍇 减速食物 (+15分，减慢蛇速)
- **完整控制**: 键盘控制 + 移动端触摸按钮
- **游戏状态显示**: 实时分数、食物统计、游戏状态
- **碰撞检测**: 墙壁碰撞和自身碰撞检测
- **暂停/继续功能**: 空格键暂停游戏

## 🚀 快速开始

1. 克隆或下载本项目
2. 直接在浏览器中打开 `index.html`
3. 点击"开始游戏"按钮开始游戏

## 🎯 游戏控制

### 键盘控制
- **方向键 (↑ ↓ ← →)**: 控制蛇的移动方向
- **空格键**: 暂停/继续游戏
- **R 键**: 重新开始游戏

### 移动端控制
- 使用屏幕上的方向按钮控制
- 开始/暂停/重新开始按钮

## 📱 游戏规则

1. 使用方向键控制蛇的移动
2. 吃到食物增加分数和蛇的长度
3. 不同食物类型有不同的效果：
   - 红色食物 🍎: 普通食物，+10分
   - 橙色食物 🍒: 加速食物，+20分，蛇速加快
   - 紫色食物 🍇: 减速食物，+15分，蛇速减慢
4. 撞到墙壁或自己的身体游戏结束
5. 尽可能获得高分！

## 🛠️ 技术栈

- **HTML5**: 页面结构
- **CSS3**: 样式和动画，使用 Flexbox 布局
- **JavaScript (ES6+)**: 游戏逻辑，使用 Canvas API
- **Canvas API**: 游戏图形渲染
